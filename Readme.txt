================================================
========	Configuracion de proyecto	========
================================================
1.-Instalar nodejs
2.-Actualizar npm, comando
	npm install -g npm@latest
3.-Verison de npm
	npm -version
4.-Limpiar cache de npm
	npm cache clean --force
5.-Descargar proyecto de git
6.-instalar dependencia de proyecto
	npm install
7.-Levantar proyecto local
	npm start
8.-Cuando se agregue una nueva dependencia, se tiene que guardar en el package-lock.json con el comando
	npm install nombreDependencia --save
